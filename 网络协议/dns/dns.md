```
/* 填充首部的格式大致相同，下面的填充值是参考他人抓包分析的结果 */
  buf[0] = 0x00;
  buf[1] = 0;
  buf[2] = 0x01;
  buf[3] = 0;
  buf[4] = 0;
  buf[5] = 1;
  buf[6] = buf[7] = 0;
  buf[8] = buf[9] = buf[10] = buf[11] = 0;
```

- 域名格式：该部分一数字开始以0结束。
- 查询类型：1代表IP地址、2代表名字服务器、5代表规范名称、12代表指针记录
- 查询类：1代表互联网
